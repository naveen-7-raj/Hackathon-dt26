📖 How to Read SHAP Values
⬆️
Positive SHAP Value
This feature pushed the model toward Approved. A larger bar = stronger positive push for this applicant.
⬇️
Negative SHAP Value
This feature pushed the model toward Rejected. A larger bar = stronger negative pull against this applicant.
📏
Mean |SHAP| (Global)
Average of absolute SHAP values across all predictions. Taller bar = feature has more overall influence on every decision the model makes.
